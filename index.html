Connect Google Sheet

Paste your Google Sheet URL. The sheet must be shared as "Anyone with the link can view".

  1. Open your Google Sheet
  2. Click Share (top right)
  3. Under "General access" → Anyone with the link → Viewer
  4. Click Copy link and paste it below
Admin Access (Optional)

Set a private admin password — only you will use this to add, edit or delete project descriptions from the dashboard.

How to set up the Apps Script
  1. Open apps-script.gs from your dashboard folder and copy all the code
  2. In your Google Sheet → Extensions → Apps Script
  3. Delete the default code, paste, press Ctrl+S to save
  4. Click Deploy → New deployment → Web app
  5. Execute as: Me  |  Who has access: Anyone
  6. Click Deploy, copy the Web App URL and paste it above
